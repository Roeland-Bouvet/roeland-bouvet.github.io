The BEM-ish Building

The building below is a demonstration on the use of the BEM convention, while allowing a bit more leniency/flexibility, with the goal of providing a clear and consistent CSS. Feel free to inspect the building below and the use of blocks, elements and modifiers. Check out the SCSS here.

Notes:
  • Avoid using very generic classnames that could be used already in other parts.
  • Avoid chaining too many "&__" elements. Look to start a new "block" class when it makes sense.
  • Define classes that can be used in several different elements/blocks higher up the tree.
